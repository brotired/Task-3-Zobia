# Random Password Generator Python Program

## Description
This is a simple Random Password Generator built using Python.

The program asks the user to enter a password length and then creates a random password using a combination of letters and numbers.

This project was created to practice:
- Importing Python modules
- Using the `random` module
- String manipulation
- Loops
- User input

## Features
- Allows users to choose password length
- Generates a random password
- Uses uppercase letters, lowercase letters, and numbers
- Creates a different password each time the program runs
